# Obsidian Notion Data Mismatch Example

This example demonstrates the core challenge discussed in the article: integrating flexible, unstructured notes (like those from Obsidian) with structured database systems (like Notion). It simulates several Obsidian-style decision notes and then attempts to extract structured data from them using regular expressions, highlighting the manual effort, potential for inconsistency, and fragility involved in bridging these two data paradigms. Finally, it shows how querying becomes easier once data is structured, but only after this initial extraction effort.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run `python main.py` in your terminal.
